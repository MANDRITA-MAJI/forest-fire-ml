# 🌲 Forest Fire Prediction

A Machine Learning web application that predicts the likelihood of forest fires based on weather and environmental metrics (temperature, humidity, wind speed, etc.).

## 🛠️ Built With
* **Python**
* **Scikit-Learn, Pandas, NumPy** (Data processing & ML)
* **Flask** (Web backend)

## 📂 Quick Structure
* `notebook/`: Data exploration (EDA) and model training experiments.
* `models/`: Saved (pickled) ML models and scalers for quick loading.
* `application.py`: The Flask web server routing the predictions.
* `template/`: HTML files for the web interface.
